About

LinguisticDB is a project that provides an extensive search interface for Project Gutenberg books. We strive to make the exploration of knowdedge and literature more accessible to all, and provide a proprietary comparison algorithm that compares books and authors from all centuries based on linguistic features.

The advanced search features of this website let users filter and compare books based on very specific characteristics.

The Statistics page provides general statistics about the LinguisticDB dataset.

We are independent from the Project Gutenberg.
